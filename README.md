🚖 RideBuddy – AI Voice Ride Booking App

RideBuddy is an AI-powered ride booking platform that allows users to book cabs with voice commands. It integrates Next.js (frontend), FastAPI (backend), MongoDB (database), and Stripe (payments) to deliver a smooth booking experience.

Features

🎤 Voice-based ride booking (speech-to-text)

🔐 User authentication with JWT (HttpOnly cookies)

🗺️ Fare calculation based on mileage and distance slabs

💳 Stripe checkout integration (secure card payments)

📧 Email invoice sending after successful payment

🛠 MongoDB storage for rides

📱 Responsive UI with Next.js 14

🎨 Clean modern design with TailwindCSS + SweetAlert2
